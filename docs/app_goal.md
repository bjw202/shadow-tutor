# 📋 English Shadow Speaking Tutor - 개발 명세서

## 프로젝트 개요

모바일 웹에서 영어 텍스트를 업로드하면, 원어민 발음으로 읽어주는 쉐도우 스피킹 학습 앱

## 기술 스택

Framework: Next.js 14+ (App Router)
Language: TypeScript
Styling: Tailwind CSS
TTS: OpenAI TTS API (alloy 또는 nova voice)
Storage: LocalStorage (진행률 저장)
Deploy: Vercel

## 핵심 기능

1. 텍스트 업로드

MD 파일 업로드 또는 텍스트 직접 붙여넣기
업로드된 텍스트를 문장 단위로 자동 분리

2. TTS 재생

OpenAI TTS API로 원어민 발음 재생

현재 읽고 있는 문장 하이라이트 표시

3. 학습 모드 (UI에서 선택)
   모드동작연속 모드전체 텍스트를 쭉 읽음쉐도잉 모드한 문장 읽고 → 대기(N초) → 다음 문장 자동 진행
4. 설정 옵션

읽기 속도: 느림 / 보통 / 빠름 (0.8x, 1.0x, 1.2x)
대기 시간 (쉐도잉 모드): 3초 / 5초 / 7초 / 10초

5. 진행률 관리

현재 문장 번호 / 전체 문장 수 표시
LocalStorage에 저장하여 앱 재접속 시 이어서 학습
수동 탐색: 이전/다음 문장 버튼
